# 部门排班系统

## 📋 项目简介

这是一个功能完善的部门排班系统，可以自动生成周末和法定节假日的值班排班表。

**特点：**
- ✅ 代码每一行都有详细注释
- ✅ 适合Python初学者学习
- ✅ 功能完整，可直接使用
- ✅ 自动生成美观的Excel排班表

## 🎯 主要功能

1. **自动识别日期**
   - 识别周末（周六、周日）
   - 识别法定节假日
   - 自动排除调休日

2. **智能排班**
   - 劳动节、国庆节分段安排
   - 连休假期固定同一组
   - 每组每月不超过4天
   - 避免连续值班

3. **接续功能**
   - 可接续上一年的排班顺序
   - 保证轮换公平性

4. **Excel导出**
   - 自动生成美观的排班表
   - 周末和节假日用不同颜色标识
   - 包含组员信息和统计数据

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

或单独安装：

```bash
pip install pandas xlsxwriter
pip install chinese_calendar
```

### 2. 运行程序

**方式一：图形界面（推荐，适合不懂代码的用户）**

双击运行 `启动排班界面.bat`，或在命令行执行：

```bash
python 排班系统界面.py
```

在界面中直接输入组员姓名，选择时间范围，点击「生成排班表」即可。

**方式二：命令行**

```bash
python 部门排班系统.py
```

### 3. 按提示输入（命令行方式）

```
请输入组数：3
请输入第1组的组员名字（用逗号分隔）：张三,李四,王五
...
请输入年份：2024
请输入起始月份：1
请输入结束月份：12
是否接续上一年的排班顺序？(y/n)：n
```

### 4. 查看结果

程序会生成Excel文件：`排班表_2024年1-12月.xlsx`

## ✨ 排班规则

1. ✅ 仅排周末和法定节假日
2. ✅ 自动排除调休日
3. ✅ 可接续上一年排班
4. ✅ 每组每月不超过4天
5. ✅ 劳动节分两段（5月1-2日、5月3-5日）
6. ✅ 国庆节分两段（10月1-3日、10月4-7日）
7. ✅ 连休假期固定同一组
8. ✅ 避免连续值班
9. ✅ 组员固定，整组值班

## 💡 常见问题

### Q1: 如何修改每月值班天数限制？

**A:** 修改代码第175行：
```python
if current_count >= 4:  # 将4改为其他数字
```

### Q2: 如何添加新的组？

**A:** 运行程序时输入更多的组数和组员即可

### Q3: 可以只排某几个月吗？

**A:** 可以，输入起始月份和结束月份时指定范围

### Q4: 如何接续上一年的排班？

**A:** 
1. 查看上一年排班表最后一行的值班组
2. 运行程序时选择"y"接续
3. 输入上一年最后值班的组号

### Q5: 调休日会被排班吗？

**A:** 不会，程序会自动识别并排除调休日
