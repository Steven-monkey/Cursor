# 部门排班系统使用说明

## 功能介绍
这是一个自动化的部门排班系统，可以帮助您生成周末和法定节假日的值班排班表。

## 安装步骤

### 1. 安装Python依赖库
在命令行中运行以下命令：
```bash
pip install -r requirements.txt
```

或者单独安装：
```bash
pip install pandas xlsxwriter
pip install chinesecalendar
```

### 2. 库的作用说明
- **pandas + xlsxwriter**: 用于导出Excel文件
- **chinesecalendar**: 用于判断中国的法定节假日

## 使用方法

### 1. 运行程序
```bash
python 部门排班系统.py
```

### 2. 按提示输入信息
程序会依次询问：
1. **组数**：输入有多少个组（例如：3）
2. **各组成员**：输入每组的成员名字，用逗号分隔（例如：张三,李四,王五）
3. **年份**：输入要排班的年份（例如：2024）
4. **起始月份**：输入开始月份（例如：1）
5. **结束月份**：输入结束月份（例如：12）
6. **是否接续上一年**：是否接续上一年的排班顺序（y/n）
   - 如果选择 y，需要输入上一年最后值班的组号
   - 如果选择 n，将从第1组开始排班

### 3. 输入示例
```
请输入组数：3

--- 第1组 ---
请输入第1组的组员名字（用逗号分隔）：张三,李四,王五

--- 第2组 ---
请输入第2组的组员名字（用逗号分隔）：赵六,钱七,孙八

--- 第3组 ---
请输入第3组的组员名字（用逗号分隔）：周九,吴十,郑十一

--- 排班时间范围 ---
请输入年份（例如：2024）：2024
请输入起始月份（1-12）：1
请输入结束月份（1-12）：12

--- 排班顺序设置 ---
是否接续上一年的排班顺序？(y/n，默认n)：y
请输入上一年最后值班的组号（1-3）：2
提示：本次排班将从第3组开始
```

## 排班规则

1. ✅ **仅排周末和节假日**：只在周六、周日和法定节假日安排值班
2. ✅ **自动排除调休日**：周末如果是调休上班日，不会安排值班
3. ✅ **接续上一年排班**：可选择接续上一年的排班顺序，保证轮换公平性
4. ✅ **每月限制**：每人每月值班不超过4天
5. ✅ **法定节假日**：法定节假日安排一个组值班
6. ✅ **特殊节假日**：
   - 劳动节：5月1-2日一组，5月3-5日一组
   - 国庆节：10月1-3日一组，10月4-7日一组
7. ✅ **连休固定组**：当法定节假日与周末连休时（如清明节、端午节、中秋节等），整个连休期间由同一个组值班
8. ✅ **组员固定**：每组人员固定，整组值班
9. ✅ **避免连续**：不同时段之间不安排连续值班，保证休息时间
10. ✅ **Excel输出**：自动生成美观的Excel排班表

## 输出文件

程序会生成一个Excel文件，文件名格式：`排班表_2024年1-12月.xlsx`

### Excel表格包含：
- 标题：显示年份和月份范围
- 组员信息：列出所有组的成员
- 排班明细：包括日期、星期、类型、值班组、值班人员
- 美化效果：
  - 周末背景色为浅黄色
  - 法定节假日背景色为浅红色
  - 表头为浅蓝色
  - 所有单元格带边框

## 程序结构说明（学习用）

### 主要函数：
1. **get_user_input()**: 获取用户输入
2. **get_all_schedule_dates()**: 找出所有需要排班的日期
3. **get_holiday_periods()**: 识别所有连续的假期时段
4. **get_period_identifier()**: 为假期时段生成唯一标识符
5. **create_schedule()**: 创建排班表的核心逻辑
6. **find_available_group_for_period()**: 为整个假期时段找到合适的值班组
7. **find_available_group()**: 为单天找到合适的值班组
8. **export_to_excel()**: 导出Excel文件
9. **main()**: 主函数，协调所有功能

## 常见问题

### Q1: 提示"ModuleNotFoundError"
**A**: 说明缺少依赖库，请运行 `pip install -r requirements.txt`

### Q2: 排班不均衡怎么办？
**A**: 程序会尽量均衡分配，但受限于每月4天上限和避免连续值班的规则，可能会有小幅差异

### Q3: 如何修改每月值班天数限制？
**A**: 在代码第175行，将 `if current_count >= 4:` 中的4改为其他数字

### Q4: 可以只排某几个月吗？
**A**: 可以，输入起始月份和结束月份时指定范围即可

### Q5: 什么时候需要接续上一年的排班？
**A**: 
- 每年年初排班时建议接续，保证各组轮换公平
- 查看上一年排班表最后一行，记录最后值班的组号
- 输入该组号，系统会从下一组开始排班

### Q6: 如何知道上一年最后是哪个组值班？
**A**: 
- 打开上一年的排班表Excel文件，查看最后一行
- 或者查看程序运行结束时显示的"最后值班"信息
- 建议每次排班后记录最后值班组号
