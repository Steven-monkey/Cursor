# 排班系统图形界面使用说明

## 适合人群

不懂代码的用户，只需在界面中输入人名，即可自动生成排班表。

## 如何启动

1. **确保已安装依赖**（首次使用需执行一次）：
   - 双击 `安装依赖.bat`
   - 或在命令行执行：`pip install -r requirements.txt`

2. **启动程序**：
   - 双击 `启动排班界面.bat`
   - 或打开命令行，进入本目录，执行：`python 排班系统界面.py`

## 使用步骤

### 第一步：设置组数和成员

- **组数**：输入有几个值班组（如 3 表示 3 组轮流值班）
- **每组组员**：在对应输入框中填写该组成员姓名，用**英文逗号**分隔  
  例如：`张三,李四,王五`

### 第二步：选择排班时间范围

- **年份**：支持 2004-2035 年（法定节假日数据随 chinesecalendar 库更新，当前通常支持到 2026 年）
- **起始月份**：如 1
- **结束月份**：如 12（表示排全年）

### 第三步：是否接续上一年（可选）

- 新年度排班时，可勾选「接续上一年的排班顺序」
- 勾选后需填写「上一年最后值班的组号」（可从上一年的排班表最后一行查看）

### 第四步：选择保存位置（可选）

- 留空则自动保存在程序所在目录
- 点击 **「浏览...」** 可选择保存到指定位置

### 第五步：生成排班表

点击 **「生成排班表」** 按钮，程序会自动：

- 识别周末和法定节假日
- 排除调休日
- 按规则分配各组值班
- 生成 Excel 排班表

生成完成后，可点击 **「打开生成的Excel」** 查看排班表。

## 排班规则说明

- 只在周末和法定节假日安排值班
- 调休日（周末但需上班）不排班
- 每组每月值班不超过 4 天
- 劳动节、国庆节等长假期会分段安排
- 连休假期由同一组值班

## 常见问题

**Q：提示找不到模块？**  
A：请先安装依赖：`pip install -r requirements.txt`

**Q：生成的 Excel 在哪里？**  
A：在程序所在目录，或您选择的保存位置。文件名格式：`排班表_2025年1-12月.xlsx`

**Q：可以修改组数吗？**  
A：可以，修改「组数」后点击「确认组数」，会刷新输入框。

**Q：如何获取最新节假日安排？**  
A：国务院一般在每年 11 月发布次年节假日安排。建议届时在命令行执行：`pip install -U chinesecalendar`，以更新节假日数据。
